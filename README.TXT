PROJECT TITLE              :           Block Game 

HOW TO START THIS PROJECT  :           Works With Left And Right Arrow Keys

RULES                      :           The game starts on pressing either 
                                       of the two arrow Keys. Your task is
                                       to eliminate all the boxes on screen
                                       by bouncing off the ball on the plank 
                                       you are provided with. the plank moves 
                                       left or right on user's instructions 
                                       through keys.
                                           The game ends if all the blocks are
                                       eliminated.
